Aplicacion de notas
Bienvenido a Aplicación de notas, una aplicación móvil desarrollada para ayudarte a
organizar tus actividades diarias de manera eficiente. Con esta aplicación, puedes crear,
editar, eliminar y marcar como favoritas tus tareas, además de gestionar tu cuenta de
usuario con Firebase Authentication. Este manual te guiará a través de todas las
funcionalidades de la aplicación para que saques el máximo provecho de ella.
